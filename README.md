# 📊 Numerical Methods Project

## 📌 Introduction

Numerical Methods are mathematical techniques used to obtain approximate solutions for complex problems where exact solutions are difficult or impossible to compute analytically.

This project focuses on the implementation of important numerical algorithms using programming to solve equations, systems of linear equations, and interpolation problems efficiently.

---

## 🎯 Objectives

* To understand the concept of numerical approximation
* To implement various numerical algorithms using programming
* To compare different methods based on accuracy and efficiency
* To apply mathematical concepts to real-world computational problems

---

## 🚀 Algorithms Implemented

### 🔹 Root Finding Methods

1. **Bisection Method**
   A simple and reliable method that repeatedly divides an interval to locate the root.

2. **Regula Falsi Method**
   Uses linear interpolation to approximate the root and improves convergence over bisection.

3. **Newton-Raphson Method**
   A fast and efficient method that uses derivatives to quickly converge to the root.

---

### 🔹 Methods for Solving Linear Equations

4. **Gauss Elimination Method**
   Converts a system of equations into upper triangular form and solves using back substitution.

5. **Gauss Seidel Method**
   An iterative technique that updates values continuously for faster convergence.

6. **Jacobi Method**
   An iterative method where all updates are done simultaneously using previous values.

---

### 🔹 Interpolation Methods

7. **Newton Forward Interpolation**
   Used for estimating values within a dataset with equal intervals.

8. **Newton Backward Interpolation**
   Applied when the required value is near the end of the dataset.
